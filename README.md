# BotVK
 Чат-бот ВКонтатке

# Настройка
 Все настройки бота находятся в файле `config.json`


 ``` js
 {
	"TOKEN": "", /* Токен сообщества
	"group_id": "", /* ID - сообщества для использования GroupPolling */
	"admins": [], /* Список ID администраторов */
	"is_explicit": true
 }
 ```

 Токен сообщества указывать в файле `.env`
 
 ``` js
 TOKEN=""
 
 ```
